# bike-sales-analytics
An interactive dashboard for analyzing bike sales and customer purchasing behavior.

## About the Dataset
This project explores customer demographics and lifestyle factors 
that influence bike purchase decisions.

## Key Variables Analyzed
- Demographics: Age, Gender, Marital Status, Income
- Lifestyle: Occupation, Education, Home Ownership, Number of Cars
- Purchase behavior: Commute Distance, Region, Bike Purchase (Yes/No)

## Tools Used
- Microsoft Excel

## Dashboard Highlights
- Sales breakdown by age group and gender
- Income vs. bike purchase trends
- Regional sales comparison (Europe vs. Pacific)
